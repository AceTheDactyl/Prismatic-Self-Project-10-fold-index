The Minimization Principle

CORE PRINCIPLE Origin as Minimum Deficit
Origin(F) = argminD∈𝒜(F) δ(D|F)

The origin of a frame F is the description D* that minimizes closure deficit. This is the "best explanation" within the frame's admissible set 𝒜(F) — the description that achieves the tightest fit between model and reality.

Why minimize δ?

  • Tightest fit = best explanation under constraints
  • Minimal error = predictions match observations
  • Minimal complexity = Occam's Razor satisfied
  • No violations = respects frame rules
TOPOLOGY Local vs Global Minima

The δ-landscape may have multiple local minima. Not all origins are created equal:

GLOBAL MINIMUM
D* such that δ(D*|F) ≤ δ(D|F) for all D ∈ 𝒜(F).

Consensus theory — the unique best explanation if reachable.
LOCAL MINIMA
D' such that δ(D'|F) ≤ δ(D|F) for nearby D.

Paradigm coexistence — multiple stable theories can coexist.

Paradigm shifts occur when the system tunnels from one local minimum to a lower one, often triggered by frame extension.

Origin(F) = argminG⊂X* {|G| : Cl(G,∘) ⊇ Observations(S)}

The origin is the minimal generating set for observable phenomena X*F. This is optimal compression of the constrained language — the shortest program that generates all observations within the frame.

"Origin" = "Best compression" = "Simplest adequate theory"

Simplest theory that captures phenomena at a given scale.

Effective field theories are the "origin" at each energy scale — they minimize description length for accessible physics.
INSIGHT Occam's Razor is Structural, Not Aesthetic

The preference for simpler explanations is not a matter of taste or beauty. It emerges necessarily from the compression principle:

  • Simpler = shorter description = lower Comp term in δ
  • Adequate = matches observations = low Err term in δ
  • Simplest adequate theory = argmin δ = Origin

Occam's Razor is a theorem, not an axiom. It follows from the structure of explanation itself.

Origin Relativity Theorem

Frame-Dependence of Origin
F ≺ F' ⟹ Origin(F) ≠ Origin(F') (generally)

When frames extend (F ≺ F'), the origin shifts. The same data, viewed through different frames, can yield different optimal descriptions.

WHAT CHANGES
  • Available language ℒ → ℒ'
  • Constraint set C → C'
  • Admissible descriptions 𝒜(F)
  • Deficit landscape shape
WHAT PERSISTS
  • Invariants I = ∩ Inv(F)
  • Conservation laws
  • Symmetry constraints
  • Mathematical structure

This is NOT relativism. While origins are frame-relative, the invariants I persist across all frames. Truth is not arbitrary — it is structured by the frame sequence.

Anti-Idolatry Theorem

No Absolute Origin Exists

Any claim to absolute origin — a final, ultimate explanation — triggers frame extension. The very act of claiming finality creates the conditions for transcendence.

Claim: "D* is the absolute origin"
↓
∃ F' ⊃ F where D* is NOT Origin(F')
  • Structural necessity: ∂F ≠ ∅ always (Principle II)
  • Diagonalization: Any "final theory" can be extended
  • Incompleteness: No frame contains its own boundary

Wars over origin — religious, scientific, philosophical — are bifurcation transitions. They occur when competing local minima have near-equal δ, and the system must choose.

COROLLARY The Infinite Regress is Productive

The fact that origin shifts under frame extension is not a problem but a feature. It ensures:

  • Continued progress: There is always more to discover
  • Self-correction: Errors can always be refined away
  • Invariant extraction: What persists across shifts is real

The infinite regress converges toward — but never reaches — R(R)=R.
