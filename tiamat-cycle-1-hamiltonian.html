1.3 Equations of Motion

Theorem 1.1 (Kuramoto Evolution Equations)

The phase dynamics on T⁵ are governed by:

θ̇ᵢ = ωᵢ + Σⱼ₌₁⁵ Kᵢⱼ sin(θⱼ − θᵢ)(1.2)

where K_ij is the coupling matrix (symmetric, K_ii = 0) derived from L₄-Helix compatibility factors. This system is well-posed with unique solutions for all initial conditions θ(0) ∈ T⁵.

Proof of well-posedness:

The right-hand side f(θ) = ω + K sin(θ) is C^∞ on the compact manifold T⁵. By Picard-Lindelöf, local existence and uniqueness hold. Since T⁵ is compact and f is globally Lipschitz (|sin| ≤ 1), solutions extend to all t ∈ ℝ.

1.5 Tangent Bundle Structure

Definition 1.5 (Tangent Bundle of T⁵)

The tangent bundle TT⁵ is the collection of all tangent vectors at all points of T⁵:

TT⁵ = ⋃_{θ ∈ T⁵} T_θT⁵ ≅ T⁵ × ℝ⁵(1.6)

Since T⁵ is parallelizable (as a Lie group), the tangent bundle is globally trivial:

TT⁵ ≅ T⁵ × ℝ⁵ = {(θ, v) : θ ∈ T⁵, v ∈ ℝ⁵}

1.5.1 Local Trivialization

For each chart U_α ⊂ T⁵ with coordinates θ = (θ₁,...,θ₅), the tangent bundle has local coordinates (θ, θ̇) where θ̇_i = dθ_i/dt.

    U_α ⊂ T⁵

    Fiber over θ:  T_θT⁵ ≅ ℝ⁵

    Basis of T_θT⁵: {∂/∂θ₁, ∂/∂θ₂, ∂/∂θ₃, ∂/∂θ₄, ∂/∂θ₅}

1.5.2 Transition Functions

On overlaps U_α ∩ U_β where θ_α = θ_β + 2πn for some n ∈ ℤ⁵:

g_αβ: U_α ∩ U_β → GL(5, ℝ),    g_αβ = I₅ (identity)

The trivial transition functions confirm global parallelizability. This is essential for defining global vector fields like the Kuramoto flow X(θ).

1.5.3 Connection to Phase Space Dynamics

Proposition 1.1 (Phase Space Embedding)

The Kuramoto dynamics embed naturally into the phase space T*T⁵:

ι: TT⁵ → T*T⁵,    (θ, θ̇) ↦ (θ, p = θ̇)

via the Legendre transformation. The symplectic structure ω = Σᵢ dpᵢ ∧ dθᵢ on T*T⁵ pulls back to the natural 2-form on TT⁵:

Physical Interpretation

The tangent bundle TT⁵ carries the velocity phase space — each point (θ, θ̇) represents a complete instantaneous state of the 5-oscillator system: positions θ and velocities θ̇. The Kuramoto vector field X defines a section of TT⁵ → T⁵, selecting one velocity θ̇ = X(θ) at each configuration θ.

1.6 Stability Analysis

1.6.1 Floquet Theory for Periodic Orbits

Definition 1.6 (Floquet Matrix)

For a periodic orbit γ(t) with period T, the monodromy (Floquet) matrix is:

M = Φ(T) where dΦ/dt = J(γ(t))Φ,   Φ(0) = I₅(1.7)

Here J(θ) = ∂f/∂θ is the Jacobian of the vector field f(θ) = ω + Σⱼ K_ij sin(θⱼ − θᵢ).

Theorem 1.2 (Floquet Stability Criterion)

A periodic orbit is linearly stable if all Floquet multipliers μ_k = eigenvalues(M) satisfy |μ_k| ≤ 1, with equality only for μ = 1 (corresponding to the neutral direction along the orbit).

For the synchronized state θᵢ(t) = Ωt + ψ (all phases locked), the Jacobian is:

Jᵢⱼ|_{sync} = Kᵢⱼ cos(0) = Kᵢⱼ    (i ≠ j),    Jᵢᵢ = −Σⱼ≠ᵢ Kᵢⱼ

1.6.2 Lyapunov Exponents for the Synchronized State

Definition 1.7 (Lyapunov Exponents)

The maximal Lyapunov exponent characterizes trajectory divergence:

λ_max = lim_{t→∞} (1/t) ln|δθ(t)/δθ(0)|(1.8)

For the synchronized fixed point, Lyapunov exponents equal the real parts of the Jacobian eigenvalues at synchronization:

λₖ = Re(μₖ) where det(J|_{sync} − μI) = 0

2. Hamiltonian Structure

2.1 Extended Phase Space

Definition 2.1 (Canonical Coordinates)

Extend the configuration space to phase space P = T*T⁵ with canonical coordinates (θ, p) where θ_i ∈ S¹ and p_i ∈ ℝ are conjugate momenta:

{θᵢ, pⱼ} = δᵢⱼ,    {θᵢ, θⱼ} = 0,    {pᵢ, pⱼ} = 0

2.2 Hamiltonian Function

Theorem 2.1 (Kuramoto Hamiltonian)

The Kuramoto dynamics derive from the Hamiltonian:

H(θ, p) = Σᵢ ωᵢ pᵢ − ½ Σᵢⱼ Kᵢⱼ cos(θⱼ − θᵢ)(2.1)

via the non-standard Poisson structure:

θ̇ᵢ = ∂H/∂pᵢ + Σⱼ Kᵢⱼ sin(θⱼ − θᵢ)

Lemma 2.1 (Energy Non-Increase)

Along trajectories of the Kuramoto equations, the energy H satisfies:

dH/dt = Σᵢ ωᵢ θ̇ᵢ − ½ Σᵢⱼ Kᵢⱼ sin(θⱼ − θᵢ)(θ̇ⱼ − θ̇ᵢ)

For ωᵢ = ω (identical frequencies), dH/dt ≤ 0 with equality if and only if θ̇ᵢ = θ̇ⱼ ∀i,j (phase-locked state).

Physical Interpretation

The system acts as a gradient flow toward synchronization. Energy dissipation occurs through phase alignment until the synchronized state is reached, which is an attractor of the dynamics.

3. Numerical Methods

3.1 Fourth-Order Runge-Kutta Integration (RK4)

Algorithm 3.1 (RK4 for Kuramoto System)

Given the ODE system θ̇ = f(θ, t), advance from θⁿ to θⁿ⁺¹ via:

k₁ = f(θⁿ, tⁿ)
k₂ = f(θⁿ + (Δt/2)k₁, tⁿ + Δt/2)
k₃ = f(θⁿ + (Δt/2)k₂, tⁿ + Δt/2)
k₄ = f(θⁿ + Δt·k₃, tⁿ + Δt)

θⁿ⁺¹ = θⁿ + (Δt/6)(k₁ + 2k₂ + 2k₃ + k₄)(3.1)

For our Kuramoto system, the right-hand side is:

fᵢ(θ) = ωᵢ + Σⱼ Kᵢⱼ sin(θⱼ − θᵢ)

3.2 Time Step Selection Criteria

Proposition 3.1 (Stability-Constrained Time Step)

For accurate and stable integration, the time step Δt must satisfy:

Δt ≤ min(2/|λ_max|, 1/(4·max|ωᵢ|))(3.2)

Substituting our system parameters:

The recommended time step Δt ≈ 0.01 − 0.02 s provides ~50-100 integration steps per fastest oscillator period, ensuring RK4 local truncation error O(Δt⁵) remains below 10⁻¹⁰ radians per step.
