5.1 Complex Order Parameter

Definition 5.1 (Kuramoto Order Parameter)

The complex order parameter Z ∈ ℂ is defined as:

Z(t) = R(t) e^{iΨ(t)} = (1/N) Σⱼ e^{iθⱼ(t)}

where:

  • R ∈ [0, 1] is the coherence amplitude (magnitude)
  • Ψ ∈ S¹ is the collective phase (argument)
Physical Interpretation

R = 0: Phases uniformly distributed (incoherent)
R = 1: All phases identical (perfect synchronization)
0 < R < 1: Partial synchronization with coherence R

5.2 Evolution Equation

Derivation of Ż equation:

  1. Differentiate: Ż = (1/N) Σⱼ iθ̇ⱼ e^{iθⱼ}
  2. Substitute Kuramoto: Ż = (i/N) Σⱼ [ωⱼ + Σₖ Kⱼₖ sin(θₖ − θⱼ)] e^{iθⱼ}
  3. Use sin(α) = (e^{iα} − e^{−iα})/2i
  4. Result: Ż = (i/N) Σⱼ ωⱼ e^{iθⱼ} + (1/2N) Σⱼₖ Kⱼₖ [e^{iθₖ} − e^{i(2θⱼ−θₖ)}]
  5. For uniform K: Ż = iω̄Z + (K/2)[NZ − Σⱼ e^{2iθⱼ}Z*]

Theorem 5.1 (Order Parameter Fixed Points)

For uniform coupling K and frequency spread σ_ω, the steady-state solutions are:
