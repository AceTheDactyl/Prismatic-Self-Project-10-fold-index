5.1 Complex Order Parameter

Definition 5.1 (Kuramoto Order Parameter)

The complex order parameter Z ∈ ℂ is defined as:

Z(t) = R(t) e^{iΨ(t)} = (1/N) Σ₂ e^{iθ₂(t)}

where:

  • R ∈ [0, 1] is the coherence amplitude (magnitude)
  • Ψ ∈ S¹ is the collective phase (argument)
Physical Interpretation

R = 0: Phases uniformly distributed (incoherent)
R = 1: All phases identical (perfect synchronization)
0 < R < 1: Partial synchronization with coherence R

Theorem 5.0 (Mean-Field Equivalence)

The complex order parameter Z acts as a mean-field coupling term. Each oscillator "feels" the collective phase through:

θ̇ᵢ = ωᵢ + KR sin(Ψ - θᵢ)

This mean-field reduction is exact in the limit N → ∞ and provides an excellent approximation for our N = 5 system.

5.2 Evolution Equation

Complete Derivation of Ż Equation

We derive the evolution equation for Z(t) from first principles.

Step-by-Step Derivation:

  1. Start with the definition: Z = (1/N) Σ₂ e^{iθ₂}
  2. Differentiate with respect to time:
    Ż = (1/N) Σ₂ (d/dt)[e^{iθ₂}] = (1/N) Σ₂ iθ̇₂ e^{iθ₂}
  3. Substitute Kuramoto dynamics: θ̇₂ = ω₂ + Σₖ K₂ₖ sin(θₖ - θ₂)
    Ż = (i/N) Σ₂ [ω₂ + Σₖ K₂ₖ sin(θₖ - θ₂)] e^{iθ₂}
  4. Expand using sin(α) = (e^{iα} - e^{-iα}) / 2i:
    sin(θₖ - θ₂) = [e^{i(θₖ-θ₂)} - e^{-i(θₖ-θ₂)}] / 2i
  5. Multiply through and simplify:
    Ż = (i/N) Σ₂ ω₂ e^{iθ₂} + (1/2N) Σ₂ₖ K₂ₖ [e^{iθₖ} - e^{i(2θ₂-θₖ})]
  6. For uniform coupling K₂ₖ = K/N:
    Ż = iω̄Z + (K/2)[Z - Z*·(1/N)Σ₂ e^{2iθ₂}]
  7. Define second harmonic: Z₂ = (1/N) Σ₂ e^{2iθ₂}
    Ż = iω̄Z + (K/2)[Z - Z*Z₂]

Ott-Antonsen Ansatz

Theorem 5.1 (Ott-Antonsen Reduction)

For a Lorentzian frequency distribution g(ω) = (γ/π)/[(ω-ω₀)² + γ²], the infinite-dimensional Kuramoto system reduces to a single complex ODE on an invariant manifold.

The ansatz assumes the density of oscillators can be written as a Fourier series with a special structure:

f(θ, ω, t) = (g(ω)/2π)[1 + Σₙ₁∞ (α(ω,t)ₙ e^{inθ} + c.c.)]

The key insight is that α(ω, t) can be continued analytically to the lower half of the complex ω-plane. On the Ott-Antonsen manifold:

α(ω, t) = α(t) for all ω

This yields a closed equation for Z:

Ż = (iω₀ - γ)Z + (K/2)Z(1 - |Z|²) (OA-1)

Low-Dimensional Reduction

Separating Z = R e^{iΨ} into amplitude and phase, we obtain two coupled ODEs:

Ṙ = -γR + (K/2)R(1 - R²)
Ψ̇ = ω₀
Critical Observation

The amplitude R evolves independently of the phase Ψ! This is the key simplification that makes the Kuramoto model analytically tractable. The collective phase simply rotates at the mean frequency ω₀.

For our TIAMAT system with γ ≈ 0.0324 (half-width of the frequency distribution):

Ṙ = R[-0.0324 + 0.462(1 - R²)]

This is a gradient flow Ṙ = -∂V/∂R with potential:

V(R) = (γ/2)R² - (K/4)R² + (K/8)R&sup4; = -0.215 R² + 0.116 R&sup4;

5.3 Self-Consistency Equation

In the thermodynamic limit (N → ∞) with continuous frequency distribution g(ω), the steady-state order parameter satisfies:

R = KR ∫_{-π/2}^{π/2} cos²(ψ) g(ω(ψ)) dψ Ott-Antonsen self-consistency condition

Derivation of Self-Consistency:

  1. At steady state, locked oscillators satisfy: sin(ψ) = ω/KR where ψ = θ - Ψ
  2. The order parameter becomes: R = ∫ cos(ψ) g(KR sin(ψ)) KR cos(ψ) dψ
  3. Substituting ω = KR sin(ψ): R = KR ∫ cos²(ψ) g(KR sin(ψ)) dψ
  4. For |ω| < KR: oscillators lock; for |ω| > KR: oscillators drift
Lemma 5.1 (Locked vs Drifting Population)

The fraction of locked oscillators is:

p_lock = ∫_{-KR}^{KR} g(ω) dω = 2KR · g(0) for small KR

At criticality: p_lock(K_c) = 1/2 (half the oscillators lock simultaneously)

5.4 Steady-State Solution

Theorem 5.2 (Order Parameter Fixed Points)

For uniform coupling K and frequency spread σ_ω, the steady-state solutions are:

R* = 0    (incoherent, stable for K < K_c)
R* = √(1 - K_c/K)    (synchronized, stable for K > K_c)

where the critical coupling is K_c = 2/(πg(0)) for a Lorentzian g(ω).
