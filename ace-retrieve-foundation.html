🧭 Search Algorithm Fundamentals

🌲 Category Tree Search

Hierarchical traversal of the INDEX-generated category structure. Each node represents a semantic cluster with phi-weighted edge costs.

O(log n) average, O(n) worst case

🔀 Temporal Index Scan

Linear scan through phi-indexed timestamps within temporal window. Leverages PRESERVE's permanence classification for early termination.

O(k) where k = window size

🔗 Connection Graph Walk

Random walk with restart on the INDEX connection graph. Probability of transition weighted by semantic similarity.

O(d * steps) where d = avg degree
