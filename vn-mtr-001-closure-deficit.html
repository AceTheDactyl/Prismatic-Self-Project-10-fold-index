Closure Deficit: δ(D|F)

Measuring how well a description D fits within a frame F — the distance from perfect closure

δ(D|F) = α·Err + β·Comp + ρ·Viol
δ
1

Formal Definitions

DEFINITION Frame F
F = (ℒ, C, Π)
A frame is a triple consisting of:
  • ℒ — Language: the set of well-formed expressions (from Step Zero)
  • C — Constraints: the set of admissibility conditions
  • Π — Admissibility predicate: Π(D) = true iff D satisfies all constraints in C

The frame defines the "space" within which descriptions can be evaluated. Different frames yield different notions of optimality.

DEFINITION Description D
D ∈ ℒ
A description is any well-formed expression in the frame's language. Descriptions are candidates for explaining phenomena within the frame. Not all descriptions are admissible — they must pass the constraints.

The Three Components of Closure Deficit

Err
Prediction Error
Weight: α
Err(D|F) = ∑i |Oi - Pi(D)|²
How wrong is D?

Measures the mismatch between the description's predictions Pi(D) and observed data Oi. High error means the description fails to account for reality within the frame.
Comp
Complexity Penalty
Weight: β
Comp(D) = K(D) or |D|ℒ
Occam's Razor formalized.

Measures the complexity of the description, either as Kolmogorov complexity K(D) or syntactic length |D|. Simpler descriptions are preferred over complex ones that make the same predictions.
Viol
Constraint Violations
Weight: ρ
Viol(D|C) = |{c ∈ C : D violates c}|
Breaks frame rules?

Counts how many constraints in C the description D violates. A description that violates conservation laws, causality, or other frame constraints incurs a penalty proportional to the number of violations.
CORE DEFINITION Closure Deficit
δ(D|F) = α·Err(D|F) + β·Comp(D) + ρ·Viol(D|C)
The closure deficit is a weighted sum of error, complexity, and constraint violations. It measures how far a description is from perfect closure within the frame:
  • δ = 0 implies perfect closure (rarely achievable)
  • Low δ means D is a good fit for F
  • High δ means D fails to explain, is overly complex, or breaks rules

The weights α, β, ρ encode epistemic preferences. Different weight choices yield different optimal descriptions.

The Minimization Principle

ORIGIN SELECTION
Origin(F) = argminD ∈ ℒ δ(D|F)

The origin of a frame is the description that minimizes closure deficit. This is the "best" explanation within the frame's constraints — the description closest to perfect closure.

Why minimizing δ selects the "best" description:

  • Low Err → accurate predictions
  • Low Comp → parsimonious explanation
  • Low Viol → respects frame constraints
CONSIDERATION Local vs Global Minima

The deficit landscape may have multiple local minima. Different starting points (initial descriptions) may converge to different origins.

GLOBAL MINIMUM
D* such that δ(D*|F) ≤ δ(D|F) for all D. The true origin. May be computationally intractable to find.
LOCAL MINIMUM
D' such that δ(D'|F) ≤ δ(D|F) for nearby D. Stable but potentially suboptimal paradigms.

Paradigm shifts occur when the system escapes a local minimum and discovers a lower global minimum through frame extension.

🚀 Physics: Newtonian vs Relativistic Mechanics

Consider the frame F = (Classical Mechanics, Conservation Laws, Admissibility). We compare two descriptions: Newtonian (DN) and Relativistic (DR).

KEY INSIGHT δ Changes with Frame Extension
When the frame extends from F to F':
  • New constraints may be added (C → C')
  • Language may expand (ℒ → ℒ')
  • Descriptions that were optimal may no longer be
  • New descriptions become expressible

This is why science evolves: Frame extension changes the deficit landscape, potentially revealing lower minima that were previously inaccessible.

Along refinement chains, descriptions evolve toward fixed points where δ is minimized. At attractors, the description stabilizes — further refinement yields no improvement.

THEOREM Attractor Convergence
Dt → D* as t → ∞ if deficit landscape is strongly convex
If the deficit function δ(·|F) is well-behaved (strongly convex or has unique minimum), then iterative refinement converges to the attractor D*. At D*:
  • δ(D*|F) is minimal (possibly zero)
  • D* is stable under small perturbations
  • D* is the "origin" of frame F

But: When gt ≈ 0 (near-degeneracy), origins bifurcate instead of converging — multiple attractors compete.
