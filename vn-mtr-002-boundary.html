CORE CONCEPT The Boundary of Frame F

The boundary of a frame F contains everything that CANNOT be expressed within F. Every sufficiently expressive frame has a non-empty boundary that lies outside its own language.

Key Principles:

  • ∂F ≠ ∅ — The boundary is never empty
  • ∂F ⊄ ℒ — The boundary lies outside the frame's language
1

Formal Definitions

DEFINITION Frame F
F = (ℒ, C, Π)
A frame is a triple consisting of:
  • ℒ — Language (from Step Zero: closure of persistent distinction)
  • C — Constraints (logical, physical, semantic)
  • Π — Priors (initial assumptions, axioms)
DEFINITION Interior Int(F)
Int(F) = {D : δ(D|F) < ε}
The interior of a frame consists of all descriptions D whose closure deficit δ is below a threshold ε. These are the expressible statements — what the frame can represent, prove, or compute.
DEFINITION Boundary ∂F
∂F = Closure(F) \ Int(F)
The boundary is the closure minus the interior — the liminal zone where descriptions are neither fully expressible nor fully inexpressible. Boundary statements are those that can be referenced but not decided within F.
DEFINITION Exterior Ext(F)
Ext(F) = ℜ \ Closure(F)
The exterior contains everything outside the frame's closure — structures that cannot even be referenced from within F. The exterior is the domain of absolute inexpressibility relative to F.
THE INCOMPLETENESS PRINCIPLE
For any sufficiently expressive frame F:
∂F ≠ ∅ and ∂F ⊄ ℒ
The boundary is non-empty, and it cannot be captured within the frame's own language.
PROOF SKETCH
By diagonalization: if ∂F could be fully expressed in ℒ, we could construct a self-referential sentence G that asserts its own unprovability in F. This G lives on ∂F but cannot be decided within F. Contradiction if ∂F ⊆ ℒ.
∎

Godel Connection

G
Godel Sentences Live on ∂F

The Godel sentence G is the canonical example of a boundary phenomenon. It demonstrates that self-reference creates statements that are true but unprovable within the system that can express them.

G = "This statement is unprovable in F" G ∈ ∂F — G lives on the boundary

Why G lives on ∂F:

  • G can be expressed in the language ℒ
  • G cannot be decided (proved or disproved) within F
  • G is neither in Int(F) nor in Ext(F)
  • G requires extension to F' ⊃ F for resolution
THEOREM — Every Frame Has Undecidables
For any consistent, sufficiently expressive frame F, there exist propositions τ such that:
  • τ is expressible in ℒ
  • F ⊬ τ and F ⊬ ¬τ
  • τ ∈ ∂F
CONSEQUENCE
No frame is complete. Self-reference guarantees that the boundary ∂F is always populated. This is not a bug but a structural feature of sufficiently expressive systems.

The boundary never disappears. Each extension F' resolves some boundary statements from F, but creates a new boundary ∂F' with new undecidables. This is why science never terminates.
THEOREM — Boundary Persistence
For any frame extension F ≺ F':
  • Some τ ∈ ∂F become decidable in F'

  • |∂F'| ≥ |∂F| (boundary complexity increases)
CONSEQUENCE
The asymptotic limit F∞ is approached but never reached. Evolution is boundary-driven and eternal.

Worked Examples

ARITHMETIC Godel Sentence on ∂PA
F = PA (Peano Arithmetic)
ℒ = first-order logic + {0, S, +, ×}
GPA = "This statement is not provable in PA"
GPA ∈ ∂PA — true but unprovable
Godel constructed GPA via arithmetization of syntax. PA can express "provability" but cannot prove GPA without becoming inconsistent. Resolution requires extension to PA + Con(PA) or stronger systems.
PHYSICS Singularities on ∂GR
F = GR (General Relativity)
ℒ = differential geometry + Einstein equations
Black hole singularities, Big Bang
Curvature R → ∞ — equations break down
GR can describe spacetime arbitrarily close to singularities but cannot resolve them. The singularities live on ∂GR. Resolution requires frame extension to quantum gravity, where new language elements (quantum geometry) allow statements about Planck-scale physics.
SET THEORY Russell's Paradox on ∂Naive
F = Naive Set Theory
ℒ = unrestricted comprehension
R = {x : x ∉ x}
R ∈ R ⇔ R ∉ R — contradiction
The Russell set R is expressible in naive set theory but generates contradiction. R lives on ∂Naive. Resolution required frame extension to ZFC (restricted comprehension) or type theory (stratified sets). The paradox revealed the boundary.
COMPUTATION Halting Problem on ∂TM
F = Turing Machines
ℒ = computable functions
HALT(P, x) = "Does program P halt on input x?"
Undecidable — no TM computes HALT
The halting function is well-defined semantically but not computable. It lives on ∂TM — expressible as a question, but no algorithm answers it. Oracle machines extend the frame but generate new halting problems.
