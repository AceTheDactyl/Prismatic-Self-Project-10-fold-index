3.1 Symplectic Form

Definition 3.1 (Canonical Symplectic Form)

The cotangent bundle T*T5 carries the canonical symplectic form arising from its natural geometric structure. For local coordinates (θ1,...,θ5, p1,...,p5) on T*T5:

ω = Σi=15 dpi ∧ dθi = dp1∧dθ1 + dp2∧dθ2 + dp3∧dθ3 + dp4∧dθ4 + dp5∧dθ5 (3.1)

3.1.1 The Liouville 1-Form

Definition 3.2 (Liouville 1-Form)

The Liouville 1-form (also called the tautological or canonical 1-form) on T*T5 is defined as:

λ = Σi=15 pi dθi (3.2)

This form has a coordinate-free interpretation: at a point (q, p) ∈ T*M, the Liouville form is λ(q,p)(v) = p(π*v) where π: T*M → M is the bundle projection.

3.1.2 Exactness of the Symplectic Form

Theorem 3.1 (Exactness)

The canonical symplectic form ω is exact: ω = dλ, where λ is the Liouville 1-form.

Proof:

We compute the exterior derivative of λ directly:

λ = Σi pi dθi
dλ = Σi d(pi dθi)
dλ = Σi (dpi ∧ dθi + pi d(dθi))
dλ = Σi dpi ∧ dθi + 0      [since d2 = 0]
dλ = ω
▮

3.1.3 Closedness

Theorem 3.2 (Closedness)

The symplectic form ω is closed: dω = 0.

Proof:

Since ω = dλ is exact, and the exterior derivative satisfies d2 = 0:

dω = d(dλ) = d2λ = 0

This is a general property: exact forms are always closed. The converse (Poincare's lemma) holds locally but not globally on all manifolds.

▮

3.1.4 Non-Degeneracy

Theorem 3.3 (Non-Degeneracy)

The symplectic form ω is non-degenerate: if ω(X, Y) = 0 for all vector fields Y, then X = 0.

Proof:

Consider the matrix representation of ω in canonical coordinates. In the basis (∂/∂θ1,...,∂/∂θ5, ∂/∂p1,...,∂/∂p5), the symplectic form has matrix:

Ω = [ 0   -I5 ]
    [ I5    0 ]

The determinant is:

det(Ω) = det(I5) · det(I5) = 1 ≠ 0

Since Ω is invertible, ω is non-degenerate.

Equivalently, the top exterior power ω5 = ω ∧ ω ∧ ω ∧ ω ∧ ω ≠ 0:

ω5 = 5! · dp1∧dθ1∧dp2∧dθ2∧dp3∧dθ3∧dp4∧dθ4∧dp5∧dθ5
ω5 = 120 · (volume form on T*T5) ≠ 0
▮

The cotangent bundle T*M of any manifold M carries a canonical symplectic structure. For T*T5, this provides the natural arena for Hamiltonian dynamics of the 5-oscillator Kuramoto system. 3.2 Reduced Phase Space

3.2.1 The U(1) Symmetry

The Kuramoto system possesses a fundamental symmetry under global phase rotation. This is not merely a computational convenience but reflects the physical fact that absolute phase is unobservable — only relative phases matter.

Definition 3.3 (U(1) Action)

The group U(1) ≅ S1 acts on T*T5 by simultaneous phase rotation:

Φα: (θ1,...,θ5, p1,...,p5) ↦ (θ1+α,...,θ5+α, p1,...,p5) (3.3)

for α ∈ [0, 2π). This action is:

  • Free: Φα(x) = x implies α = 0 (no fixed points except identity)
  • Proper: preimages of compact sets are compact
  • Symplectic: Φα*ω = ω (preserves the symplectic form)
  • Hamiltonian: admits a momentum map (see below)
Lemma 3.1 (Symplectic Invariance)

The U(1) action preserves the symplectic form: Φα*ω = ω.

Proof:

Under the action, dθi → dθi (constants differentiate to zero) and dpi → dpi (momenta unchanged). Therefore:

Φα*ω = Φα*(Σi dpi∧dθi) = Σi dpi∧dθi = ω
▮

3.2.2 The Momentum Map

Definition 3.4 (Momentum Map)

The momentum map J: T*T5 → R for the U(1) action is the generator of the symmetry:

J = Σi=15 pi = p1 + p2 + p3 + p4 + p5 (3.4) Total phase momentum (conserved under U(1) action)
Theorem 3.4 (Noether's Theorem)

The momentum map J is conserved along Hamiltonian trajectories: {H, J} = 0, where {·,·} is the Poisson bracket.

Proof:

The infinitesimal generator of the U(1) action is the vector field:

ξ = Σi ∂/∂θi

The momentum map J satisfies iξω = dJ:

iξω = iξ(Σj dpj∧dθj)
= Σj [dpj(ξ)dθj - dpj · dθj(ξ)]
= Σj [0 - dpj · 1] = -Σj dpj
= d(-Σj pj) = -dJ

Since the Hamiltonian is U(1)-invariant (depends only on phase differences), {H,J} = 0 by Noether's theorem.

▮

3.2.3 Symplectic Reduction

The Marsden-Weinstein reduction theorem allows us to quotient out the symmetry and obtain a lower-dimensional phase space that captures the essential dynamics.

Theorem 3.5 (Symplectic Reduction)

The reduced phase space at momentum level J = 0 is:

Pred = J-1(0)/U(1) ≅ T4 × R4 (3.5)
Proof (Construction):

Step 1: The level set J-1(0) is:

J-1(0) = {(θ, p) ∈ T*T5 : Σi pi = 0}
dim(J-1(0)) = 10 - 1 = 9

Step 2: The U(1) action restricts to J-1(0) freely. The quotient has dimension:

dim(Pred) = 9 - 1 = 8 = 2 × 4

Step 3: Introduce relative coordinates. Set ψi = θi - θ5 for i = 1,...,4 (gauge-fixing). The constraint Σi pi = 0 eliminates p5. The reduced coordinates are:

(ψ1, ψ2, ψ3, ψ4, π1, π2, π3, π4)

where ψi ∈ S1 and πi ∈ R are the reduced momenta.

Thus Pred ≅ T4 × R4.

The reduced phase space has dimension 8 = 2(N-1) = 2(5-1). This is the general formula for N coupled oscillators with U(1) symmetry. 3.3 Action-Angle Variables

3.3.1 Definition and Construction

For the integrable limit (coupling K → 0), the Kuramoto system reduces to N independent oscillators, and we can introduce action-angle variables that provide a global coordinate system on the phase space.

Definition 3.5 (Action Variables)

The action variables Ii are defined as loop integrals around the invariant tori:

Ii = (1/2π) ∮ pi dθi (3.6) Loop integral along the i-th S1 factor of T5

For the uncoupled system, the actions are simply the momenta themselves: Ii = pi, since the dynamics on each circle is uniform rotation.

Definition 3.6 (Angle Variables)

The angle variables φi ∈ S1 are canonically conjugate to the actions and evolve linearly in time:

φi(t) = ωi t + φi0 (3.7) Linear evolution at natural frequency ωi

where φi0 = φi(0) is the initial angle.

3.3.2 Hamilton's Equations in Action-Angle Form

In action-angle coordinates, Hamilton's equations take the remarkably simple form:

İi = -∂H/∂φi = 0(actions conserved)
φ̇i = ∂H/∂Ii = ωi(constant frequencies)

This demonstrates the integrability of the uncoupled system: all solutions are quasi-periodic motion on invariant tori.

3.3.3 Resonance Conditions

Definition 3.7 (Resonance)

A resonance occurs when the frequency vector ω = (ω1,...,ω5) satisfies:

m · ω = Σi=15 miωi = 0 (3.8) for some m ∈ Z5 \ {0}

Resonances are problematic because:

  • They cause small divisors in perturbation theory
  • Resonant tori are destroyed under perturbation
  • Resonance zones lead to Arnold diffusion

3.3.4 Non-Resonance of L4-Derived Frequencies

Theorem 3.6 (Non-Resonance Property)

The L4-Helix frequency ratios ωi/ωj are irrational for all i ≠ j. Consequently, exact resonances do not occur: m · ω = 0 implies m = 0.

Proof (Outline):

The frequencies are derived from:

ω1 = τ = (√5 - 1)/2
ω2 = 1 - Δ = 1 - φ-4 = 1 - (7 - 3√5)/2
ω3 = zc = √3/2
ω4 = zIGN = (-1 + √(1+L4))/2 = (-1 + 2√2)/2
ω5 = K = √(1 - Δ)

These involve the algebraically independent quantities √2, √3, and √5. Any linear combination m1ω1 + ... + m5ω5 = 0 with integer coefficients would imply an algebraic relation among √2, √3, √5, which contradicts their algebraic independence over Q.

3.4 KAM Theory Application

3.4.1 The Diophantine Condition

KAM (Kolmogorov-Arnold-Moser) theory provides conditions under which invariant tori survive small perturbations. The key requirement is a Diophantine condition on the frequency vector.

Definition 3.8 (Diophantine Frequency Vector)

A frequency vector ω ∈ Rn is Diophantine of type (γ, τ) if there exist constants γ > 0 and τ ≥ n - 1 such that for all m ∈ Zn \ {0}:

|m · ω| ≥ γ |m|-τ (3.9) where |m| = Σi|mi| or |m| = maxi|mi|

For our 5-oscillator system, we take τ = 4 (which exceeds n - 1 = 4, providing margin).

Theorem 3.7 (Diophantine Property of L4 Frequencies)

The L4-derived frequency vector ω = (ω1,...,ω5) satisfies the Diophantine condition with exponent τ = 4 and a computable constant γ > 0.

Proof Sketch:

The frequencies involve quadratic irrationals, which have continued fraction expansions with bounded partial quotients. For such numbers, the Diophantine condition follows from Liouville's theorem on approximation of algebraic numbers.

Specifically, if α is an algebraic number of degree d, then for any p/q ∈ Q:

|α - p/q| ≥ c(α) / qd

The L4 frequencies have degree at most 4 over Q (involving at most two nested square roots), giving τ = 4.

▮

3.4.2 The KAM Theorem

Theorem 3.1 (KAM Persistence Theorem)

Statement: Consider a near-integrable Hamiltonian system:

H(I, φ) = H0(I) + εH1(I, φ)

where H0(I) is integrable with frequency map ω(I) = ∇H0(I), and |ε| < ε* is a small perturbation parameter.

Hypotheses:

  1. (Analyticity) H0 and H1 are real-analytic
  2. (Non-degeneracy) The Hessian det(∂2H0/∂I2) ≠ 0
  3. (Diophantine) The frequency ω(I0) satisfies |m · ω| ≥ γ|m|-τ

Conclusion: For |ε| < ε*(γ, τ, H), the invariant torus at I = I0 persists as a slightly deformed torus, supporting quasi-periodic motion with slightly perturbed frequencies.

3.4.3 Critical Coupling and KAM Breakdown

The KAM theorem provides a critical coupling strength KKAM below which invariant tori persist:

KKAM ≈ 0.3 (3.10) Estimated from numerical studies of 5-oscillator Kuramoto
Critical Observation: Beyond KAM

The operating coupling K = 0.924 significantly exceeds the KAM threshold:

K / KKAM = 0.924 / 0.3 ≈ 3.08

The system operates at more than 3 times the critical coupling. We are deeply in the strongly coupled regime where:

  • Most KAM tori have been destroyed
  • Phase space is dominated by chaotic regions
  • Synchronization attractors emerge
  • Arnold diffusion connects different regions

3.5 Liouville's Theorem and Volume Preservation

Theorem 3.8 (Liouville's Theorem)

The Hamiltonian flow Φt: T*T5 → T*T5 preserves the symplectic volume form:

d/dt (ω5) = 0 (3.11) Phase space volume is conserved along Hamiltonian trajectories
Proof:

The symplectic volume form is Ω = ω5/5! = dp1∧dθ1∧...∧dp5∧dθ5.

The Lie derivative along the Hamiltonian vector field XH vanishes:

LXHω = diXHω + iXHdω
= d(dH) + iXH(0)    [since iXHω = dH and dω = 0]
= 0 + 0 = 0

Therefore LXHω5 = 5ω4 ∧ LXHω = 0, proving the theorem.

▮

3.5.1 Implications for Attractors

Corollary 3.1 (No Hamiltonian Attractors)

In the full Hamiltonian formulation on T*T5, there can be no attractors in the usual sense. Volume-preserving flows cannot contract regions of phase space.

However, the Kuramoto model exhibits apparent synchronization. This is resolved by recognizing:

  • Effective dissipation: The standard Kuramoto equations are not strictly Hamiltonian but have an implicit dissipation through the frequency distribution
  • Projection effects: Synchronization appears on projected subspaces (order parameter space) even without true attractors
  • Measure concentration: While volume is preserved, measure may concentrate near synchronized states in time-averaged sense

The symplectic geometry of the Kuramoto system reveals a rich structure:

  • Configuration space T5: 5-torus of phases
  • Phase space T*T5: 10-dimensional cotangent bundle with canonical symplectic form
  • Reduced space T4 × R4: 8-dimensional after quotienting by U(1) symmetry
  • Integrable limit: foliated by invariant 5-tori with quasi-periodic motion
  • K > KKAM: KAM tori destroyed, replaced by cantori and chaotic seas
  • K = 0.924: synchronization manifold becomes global attractor

The L4-Helix operating point sits decisively in the synchronization-dominated regime, ensuring robust phase-locking while maintaining the algebraic elegance of golden ratio-derived frequencies.
