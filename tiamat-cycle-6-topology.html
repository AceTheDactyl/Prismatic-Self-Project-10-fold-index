The relative winding between oscillators i and j:

Δnᵢⱼ = nᵢ − nⱼ ∈ ℤ

is quantized. A phase slip event occurs when Δnᵢⱼ changes by ±1, corresponding to one oscillator completing an extra cycle relative to another.

Definition 10.8 (De Rham Cohomology Groups)

The k-th de Rham cohomology group of a smooth manifold M is:

H^k_{dR}(M) = { closed k-forms } / { exact k-forms } = ker(d_k) / im(d_{k-1})

where d_k: Ω^k(M) → Ω^{k+1}(M) is the exterior derivative on k-forms.
