9. Renormalization Group Analysis

9.1 Coarse-Graining Procedure

Define block-averaged variables at scale ℓ by integrating out short-wavelength modes:

ψ_ℓ(x) = ∫ d³y W_ℓ(x − y) ψ(y) where W_ℓ is a Gaussian window of width ℓ

9.2 Flow Equations

Theorem 9.1 (RG Beta Functions)

Under rescaling x → x/b, ψ → b^{d_ψ}ψ, the couplings flow as:

dK/dℓ = (2 − d)K + β_K(K, Δ)
dΔ/dℓ = (4 − d)Δ + β_Δ(K, Δ)

In d = 3 dimensions, with one-loop corrections:

  • β_K = −K + c_K K²/Δ
  • β_Δ = Δ + c_Δ Δ²

The two-point correlation function at criticality:

⟨ψ(x)ψ*(y)⟩ ∼ exp(−|x − y|/ξ) with ξ = σ|K − K_c|^{−ν} and ν = 1/2
