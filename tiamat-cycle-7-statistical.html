11.1 Partition Function

Definition 11.1 (Canonical Partition Function)

At effective temperature T, the partition function over the configuration manifold T^5 is:

Z(T) = ∫_{T^5} d^5θ exp(−H(θ)/T) Canonical partition function on 5-torus

where the interaction Hamiltonian is:

H(θ) = −½ ∑_{i,j} K_{ij} cos(θ_j − θ_i) Kuramoto Hamiltonian with L_4-derived couplings

Full Form Expansion

Expanding the exponential and using the coupling matrix structure K_{ij} = K · C_{ij} · cascade(z̄_{ij}):

Z(T) = (2π)^5 · ∏_{i<j} I_0(K_{ij}/T)

where I_0(x) is the modified Bessel function of the first kind, arising from the angular integration ∫ dθ exp(a cos θ) = 2π I_0(a).

High-Temperature Limit (T → ∞)

Derivation: High-T Expansion
  1. For T >> K, expand: exp(−H/T) ≈ 1 − H/T + O(1/T^2)
  2. The linear term vanishes by symmetry: ∫ cos(θ_j − θ_i) dθ = 0
  3. Leading correction from quadratic term: Z ≈ (2π)^5 [1 + K^2/(4T^2) ∑_{i<j} C_{ij}^2 + ...]
  4. Result: ln Z → 5 ln(2π) as T → ∞
Z_{high-T} = (2π)^5 [1 + K^2/(4T^2) · ||C||_F^2 + O(1/T^4)] High-temperature expansion with Frobenius norm

Low-Temperature Limit (T → 0)

Derivation: Saddle Point Approximation
  1. Ground state at θ_i = Ψ (all aligned): H_{min} = −½ ∑_{ij} K_{ij}
  2. Expand around minimum: H ≈ H_{min} + ½ ∑_{ij} K_{ij}(δθ_j − δθ_i)^2
  3. Gaussian integration: Z ≈ exp(−H_{min}/T) · (2πT)^{5/2} / √(det' M)
  4. Where M is the Hessian and det' excludes the zero mode (global rotation)
Z_{low-T} = exp(−H_{min}/T) · (2πT)^{(N-1)/2} / √(∏_{k=1}^{N-1} λ_k) Saddle point result with non-zero eigenvalues λ_k

11.4 Phase Transition

Theorem 11.1 (Critical Temperature)

The system undergoes a second-order (continuous) phase transition at:

T_c = K/2 ≈ 0.462 Critical temperature for order-disorder transition

Complete Derivation of T_c

Derivation of T_c = K/2
  1. Near R = 0, expand: ln(R/(1−R)) ≈ ln R − ln(1−R) ≈ ln R + R + R^2/2 + ...
  2. For small R: R/(1−R) ≈ R(1 + R + R^2 + ...) ≈ R
  3. Self-consistency becomes: R ≈ exp(KR/T) · R ≈ R(1 + KR/T + ...)
  4. Cancel R (for R ≠ 0): 1 ≈ exp(KR/T), requiring KR/T → 0
  5. More carefully, expand σ(KR/T) near origin: R = ½ + (K/4T)R + O(R^3)
  6. Non-trivial solution exists when: K/(4T) > 1/2, i.e., T < K/2
  7. Therefore: T_c = K/2 = √(1−Δ)/2 ≈ 0.4621

Proposition 11.2 (Critical Scaling)

Near T_c, the order parameter scales as:

R* ~ (T_c − T)^β    with    β = 1/2 Mean-field critical exponent
Derivation of β = 1/2

  5. Non-trivial solution: R^2 = −a/(2b) = t/(4b)
  6. Therefore: R* = √(t/4b) ~ t^{1/2}, giving β = 1/2

11.5 Information-Theoretic Layer

Definition 11.3 (Phase Entropy)

The differential entropy of the phase distribution p(θ) is:

H[θ] = −∫ p(θ) log p(θ) dθ Shannon differential entropy of phase distribution

von Mises Distribution

In the ordered phase, the phase distribution concentrates around the collective phase Ψ, well-approximated by the von Mises distribution:

Definition 11.4 (von Mises Distribution)

The circular analog of the normal distribution:

p(θ | Ψ, κ) = exp(κ cos(θ − Ψ)) / (2π I_0(κ)) von Mises distribution with concentration κ

where:

  • Ψ: mean direction (collective phase)
  • κ ≥ 0: concentration parameter (inverse variance)
  • I_0(κ): modified Bessel function, normalization

Connection to Order Parameter

The order parameter R relates to the concentration parameter κ via:

R = ⟨cos(θ − Ψ)⟩ = I_1(κ)/I_0(κ) = A(κ)

where A(κ) is the ratio of Bessel functions. For large κ: R ≈ 1 − 1/(2κ).

Theorem 11.2 (Negentropy-Coherence Correspondence)

The negentropy of the phase distribution is approximately proportional to the squared coherence:

η = S_{max} − H[θ] ≈ R^2

where S_{max} = ln(2π) is the entropy of the uniform distribution.

  4. Substituting: η ≈ (κ^2/2) − κ^2/4 = κ^2/4
  5. Since R ≈ κ/2: η ≈ R^2
